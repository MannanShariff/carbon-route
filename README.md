# 🌱 CarbonIQ - Eco Route Finder

AI-based route recommendation system that suggests the lowest carbon emission travel options.

## Features
- Multiple route comparison
- CO₂ emission prediction (ML model)
- Smart suggestions based on user type
- Interactive map visualization

## Tech Stack
- Python
- Streamlit
- OpenStreetMap API
- Machine Learning (Scikit-learn)

## Run Locally
streamlit run backend/app.py
